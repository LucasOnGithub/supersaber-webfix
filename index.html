An open source clone of Beat Saber with 6000 songs

Play now for free on The Browser for the the VR Internet

We are huge fans of Beat Saber and what it has done for VR. As an homage, we built using A-Frame that runs on any browser.

We've open sourced the code and assets for free for everyone to learn.
